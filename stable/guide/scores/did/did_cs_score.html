As in the description of the DiD model, the required nuisance elements are

\[\begin{split}\begin{align} g^{\text{treat}}_{0,\mathrm{g}, t, \text{eval} + \delta}(X_i) &:= \mathbb{E}[Y_{i,t} |X_i, G_i^{\mathrm{g}}=1, T_i=t], \\ g^{\text{control}}_{0,\mathrm{g}, t, \text{eval} + \delta}(X_i) &:= \mathbb{E}[Y_{i,t} |X_i, C_{i,t_\text{eval} + \delta}^{(\cdot)}=1, T_i=t], \\ m_{0, \mathrm{g}, t_\text{eval} + \delta}(X_i) &:= P(G_i^{\mathrm{g}}=1|X_i, G_i^{\mathrm{g}} + C_{i,t_\text{eval} + \delta}^{(\cdot)}=1). \end{align}\end{split}\]

for \(t\in\{t_\text{pre}, t_\text{eval}\}\) and a certain choice of \((\mathrm{g}, t_\text{pre}, t_\text{eval})\) and \(\delta\) and control group \(C_{i,t_\text{eval} + \delta}^{(\cdot)}\).

For notational purposes, we will omit the subscripts \(\mathrm{g}, t_\text{pre}, t_\text{eval}, \delta\) in the following and use the notation

  • \(g_0(1, 0, X_i) \equiv g^{\text{treat}}_{0,\mathrm{g}, t_\text{pre}, \text{eval} + \delta}(X_i)\) (pop. outcome regr. function for treatment group in \(t_\text{pre}\))

  • \(g_0(1, 1, X_i) \equiv g^{\text{treat}}_{0,\mathrm{g}, t_\text{eval}, \text{eval} + \delta}(X_i)\) (pop. outcome regr. function for treatment group in \(t_\text{eval}\))

  • \(g_0(0, 0, X_i) \equiv g^{\text{control}}_{0,\mathrm{g}, t_\text{pre}, \text{eval} + \delta}(X_i)\) (pop. outcome regr. function for control group in \(t_\text{pre}\))

  • \(g_0(0, 1, X_i) \equiv g^{\text{control}}_{0,\mathrm{g}, t_\text{eval}, \text{eval} + \delta}(X_i)\) (pop. outcome regr. function for control group in \(t_\text{eval}\))

  • \(m_0(X_i)\equiv m_{0, \mathrm{g}, t_\text{eval} + \delta}(X_i)\) (generalized propensity score).

All scores in the multi-period setting have the form

\[\begin{split}\psi(W_i,\theta, \eta) := \begin{cases} \tilde{\psi}(W_i,\theta, \eta) & \text{for } G_i^{\mathrm{g}} \vee C_{i,t_\text{eval} + \delta}^{(\cdot)}=1 \\ 0 & \text{otherwise} \end{cases}\end{split}\]

i.e. the score is only non-zero for units in the corresponding treatment group \(\mathrm{g}\) and control group \(C_{i,t_\text{eval} + \delta}^{(\cdot)}\).

For the difference-in-differences model implemented in DoubleMLDIDMulti one can choose between score='observational' and score='experimental'.

score='observational' implements the score function (dropping the unit index \(i\)):

\[ \begin{align}\begin{aligned}\tilde{\psi}(W,\theta,\eta) :=\; & - \frac{G^{\mathrm{g}}}{\mathbb{E}_n[G^{\mathrm{g}}]}\theta + \frac{G^{\mathrm{g}}}{\mathbb{E}_n[G^{\mathrm{g}}]}\Big(g(1,1,X) - g(1,0,X) - (g(0,1,X) - g(0,0,X))\Big)\\& + \frac{G^{\mathrm{g}}T}{\mathbb{E}_n[G^{\mathrm{g}}T]} (Y - g(1,1,X))\\& - \frac{G^{\mathrm{g}}(1-T)}{\mathbb{E}_n[G^{\mathrm{g}}(1-T)]}(Y - g(1,0,X))\\& - \frac{m(X) (1-G^{\mathrm{g}})T}{1-m(X)} \mathbb{E}_n\left[\frac{m(X) (1-G^{\mathrm{g}})T}{1-m(X)}\right]^{-1} (Y-g(0,1,X))\\& + \frac{m(X) (1-G^{\mathrm{g}})(1-T)}{1-m(X)} \mathbb{E}_n\left[\frac{m(X) (1-G^{\mathrm{g}})(1-T)}{1-m(X)}\right]^{-1} (Y-g(0,0,X))\\=\; &\tilde{\psi}_a(W; \eta) \theta + \tilde{\psi}_b(W; \eta)\end{aligned}\end{align} \]

where the components of the final linear score \(\psi\) are

\[ \begin{align}\begin{aligned}\psi_a(W; \eta) &= \tilde{\psi}_a(W; \eta) \cdot \max(G^{\mathrm{g}}, C^{(\cdot)}),\\\psi_b(W; \eta) &= \tilde{\psi}_b(W; \eta) \cdot \max(G^{\mathrm{g}}, C^{(\cdot)})\end{aligned}\end{align} \]

and the nuisance elements \(\eta=(g, m)\).

Note

Remark that \(1-G^{\mathrm{g}}=C^{(\cdot)}\) if \(G^{\mathrm{g}} \vee C_{t_\text{eval} + \delta}^{(\cdot)}=1\).

If in_sample_normalization='False', the score is set to

\[ \begin{align}\begin{aligned}\tilde{\psi}(W,\theta,\eta) :=\; & - \frac{G^{\mathrm{g}}}{p}\theta + \frac{G^{\mathrm{g}}}{p}\Big(g(1,1,X) - g(1,0,X) - (g(0,1,X) - g(0,0,X))\Big)\\& + \frac{G^{\mathrm{g}}T}{p\lambda} (Y - g(1,1,X))\\& - \frac{G^{\mathrm{g}}(1-T)}{p(1-\lambda)}(Y - g(1,0,X))\\& - \frac{m(X) (1-G^{\mathrm{g}})T}{p(1-m(X))\lambda} (Y-g(0,1,X))\\& + \frac{m(X) (1-G^{\mathrm{g}})(1-T)}{p(1-m(X))(1-\lambda)} (Y-g(0,0,X))\\=\; &\tilde{\psi}_a(W; \eta) \theta + \tilde{\psi}_b(W; \eta)\end{aligned}\end{align} \]

with \(\eta=(g, m, p, \lambda)\), where \(p_0 = \mathbb{E}[G^{\mathrm{g}}]\) and \(\lambda_0 = \mathbb{E}[T]\) are estimated on the subsample. Remark that this will result a similar score, but just uses slightly different normalization.

score='experimental' assumes that the treatment probability is independent of the covariates \(X\) and implements the score function:

\[ \begin{align}\begin{aligned}\tilde{\psi}(W,\theta,\eta) :=\; & - \theta + \Big(g(1,1,X) - g(1,0,X) - (g(0,1,X) - g(0,0,X))\Big)\\& + \frac{G^{\mathrm{g}}T}{\mathbb{E}_n[G^{\mathrm{g}}T]} (Y - g(1,1,X))\\& - \frac{G^{\mathrm{g}}(1-T)}{\mathbb{E}_n[G^{\mathrm{g}}(1-T)]}(Y - g(1,0,X))\\& - \frac{(1-G^{\mathrm{g}})T}{\mathbb{E}_n[(1-G^{\mathrm{g}})T]} (Y-g(0,1,X))\\& + \frac{(1-G^{\mathrm{g}})(1-T)}{\mathbb{E}_n[(1-G^{\mathrm{g}})(1-T)]} (Y-g(0,0,X))\\=\; &\tilde{\psi}_a(W; \eta) \theta + \tilde{\psi}_b(W; \eta)\end{aligned}\end{align} \]

where the components of the final linear score \(\psi\) are

\[ \begin{align}\begin{aligned}\psi_a(W; \eta) &= \tilde{\psi}_a(W; \eta) \cdot \max(G^{\mathrm{g}}, C^{(\cdot)}),\\\psi_b(W; \eta) &= \tilde{\psi}_b(W; \eta) \cdot \max(G^{\mathrm{g}}, C^{(\cdot)})\end{aligned}\end{align} \]

and the nuisance elements \(\eta=(g, m)\).

Analogously, if in_sample_normalization='False', the score is set to

\[ \begin{align}\begin{aligned}\tilde{\psi}(W,\theta,\eta) :=\; & - \theta + \Big(g(1,1,X) - g(1,0,X) - (g(0,1,X) - g(0,0,X))\Big)\\& + \frac{G^{\mathrm{g}}T}{p\lambda} (Y - g(1,1,X))\\& - \frac{G^{\mathrm{g}}(1-T)}{p(1-\lambda)}(Y - g(1,0,X))\\& - \frac{(1-G^{\mathrm{g}})T}{(1-p)\lambda} (Y-g(0,1,X))\\& + \frac{(1-G^{\mathrm{g}})(1-T)}{(1-p)(1-\lambda)} (Y-g(0,0,X))\\=\; &\tilde{\psi}_a(W; \eta) \theta + \tilde{\psi}_b(W; \eta)\end{aligned}\end{align} \]

with \(\eta=(g, m, p, \lambda)\), where \(p_0 = \mathbb{E}[G^{\mathrm{g}}]\) and \(\lambda_0 = \mathbb{E}[T]\) are estimated on the subsample. Remark that this will result in a similar score, but just uses slightly different normalization.
